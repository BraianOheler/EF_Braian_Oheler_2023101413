En la carpeta static se encuentra el CSS utilizado y también así las imagenes utilizadas en la página web
En la carpeta de templates esta todo el codigo html
El app.py es el codigo princilap de Python con el framework de Flask
El db_connection.py es la conexión a la BD de My SQL

Para la ejecución es necesario en la terminal escribir: python app.py 
Y ingresar a la dirección Running on http://127.0.0.1:5000
